# Contexte de l'académie

Déposez ici les fichiers Markdown qui décrivent la mission, la vision, les offres et le ton de l'académie.

Le générateur lit automatiquement tous les fichiers `*.md` de ce dossier et les injecte dans le prompt IA.

Fichiers recommandés :

- `mission.md` : raison d'être de l'académie, promesse pédagogique, valeurs.
- `vision.md` : ambition à 1-3 ans, positionnement, différence face aux concurrents.
- `offres.md` : cours proposés, prix si souhaité, cours d'essai, test de niveau, modalités.
- `audience.md` : profils des élèves, douleurs, objectifs, objections fréquentes.
- `tonalite.md` : style de communication, mots à utiliser, mots à éviter, exemples de posts aimés.

Vous pouvez commencer uniquement avec `mission.md` et `vision.md`; les autres fichiers sont optionnels.
